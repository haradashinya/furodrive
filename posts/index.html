I started to blog in English.

I adopt HUGO to create a blog. HUGO is a static site generators. I impressed the speed of compile. And I upload posts to github.

Basic Usage

Create a post

hugo new posts/xxx.md

Preview

hugo server -D

Build

hugo -D

Deploy

cd public/
git init
git remote add origin git@github.com:xxx
git checkout -b gh-pages
git add --all
git commit -m 'initial commit'
git push origin gh-pages
